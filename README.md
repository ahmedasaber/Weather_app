# Weather-app

A new Weather-app project.

## 🚀 Feature

- Get the Current Weather for your location
- Get the 5-days Forecast for your location

## 🛠️ Technologies Used

- BLoC State Management
- MVVM Architecture
- HTTP for API requests
- Reusable-Component
- Custom Font
